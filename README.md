# 📸 Photobooth Application

A cute and interactive photobooth web application built using modern web technologies.
This project allows users to capture photos, apply fun filters, and create a simple photobooth experience directly from their browser.

## ✨ Features

* 📷 Camera access
* 🎨 Cute photo filters
* 💾 Download captured images
* 🖥️ Works on desktop and laptop
* ⚡ Simple and user-friendly UI
* 🔒 No login required

## 🚀 Tech Stack

* HTML
* CSS
* JavaScript

## 📂 Project Setup

1. Clone the repository

```bash
git clone https://github.com/Sanjukttaa/Photobooth.git
```

2. Open the project folder

```bash
cd Photobooth
```

3. Run the project

Simply open the `index.html` file in your browser.
